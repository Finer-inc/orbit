## データフロー

```
User (Browser)
  │
  │ 画面を開く
  ▼
Frontend ──REST ポーリング（実装済）──→ TS World Server (localhost:3001)
                                       │
                                       ├─→ 精霊の位置・状態を配信
                                       ├─→ 会話イベントを配信
                                       └─→ 時間帯変更を配信

Spirit (Go goroutine)
  │
  │ 定期的にtick
  ▼
AgentLoop (Go Process)
  │
  │                        HTTP localhost:3001
  ├─ observe ─────────────────→ TS World Server ──→ 視界内オブジェクト + 近くの精霊
  ├─ move_to ─────────────────→ TS World Server ──→ 位置更新 ──→ RESTで取得可
  ├─ say     ─────────────────→ TS World Server ──→ 空間ブロードキャスト
  │
  │                        Go Process内で完結
  ├─ set_goal──→ 内部目標を設定
  └─ rest    ──→ 休止状態に移行
```
